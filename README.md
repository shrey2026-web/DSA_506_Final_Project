# DSA_506_Final_Project_by_Shreyasee
# TOPIC: Can AI Understand Human Emotion Through Music?

## Research Question
Can song data, including lyrics, audio features, and artist characteristics, help AI identify emotional patterns in modern music?

## Project Description
This project analyzes Spotify music data using numerical audio features, categorical genre and artist information, and text-based lyric sentiment analysis to explore emotional patterns in modern music.

## Tools Used
- Python
- Google Colab
- Pandas
- Plotly
- Seaborn
- TextBlob
- Tableau

## Dataset
Spotify songs dataset containing:
- audio features
- lyrics
- artist information
- genre categories
- popularity metrics

## Project Components
- Data cleaning and preprocessing
- Exploratory data analysis
- Sentiment analysis
- Emotional mood classification
- Animation and storytelling visualization
- Interactive Tableau dashboard
